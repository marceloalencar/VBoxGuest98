# VBoxGuest98
VirtualBox Guest Additions backport for Microsoft Windows 98, using Windows Driver Model 1.0.

## Components
**VBoxGst**: PCI driver for VirtualBox Guest Device (`PCI\VEN&80EE&DEV_CAFE`)
